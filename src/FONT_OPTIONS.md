### Current Typography Settings:
- **Display/Poetic**: 
  - Light weight (300) for elegance
  - Letter-spacing `-0.01em` for tighter elegant look
  - Large sizes (2-4rem fluid)
- **Headings**: Letter-spacing `-0.01em`, weight 500
- **Body**: Default spacing, optimized for readability
- **UI/Buttons**: 
  - Uppercase transformation
  - Letter-spacing `0.05em` (wider for pixel fonts)
  - Weight 500
  - Labels smaller (0.875rem) with uppercase

## How to Customize

### Change Poetic Display Font:
1. Update the Google Fonts link in `/index.html`
2. Update `--font-display` in `/styles/globals.css`

### Change Heading Font:
1. Update the Google Fonts link in `/index.html`
2. Update `--font-heading` in `/styles/globals.css`

### Change Pixelated UI Font:
1. Update the Google Fonts link in `/index.html`
2. Update `--font-ui` in `/styles/globals.css`

### Remove Uppercase from Buttons:
In `/styles/globals.css`, remove or comment out:
```css
text-transform: uppercase;
```
